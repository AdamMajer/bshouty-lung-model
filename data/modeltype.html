You have the option of working with a "Single" or a "Double" lung model. In either case, both models can represent the whole pulmonary circulation. What a given model represents is based on model calibration. All default models were calibrated with the following targets:

  • Subject is 175 cm high and weighs 75 kg
  • Lung height is 20 cm with normal lung compliance
  • Measurements are performed at end-expiration (Ppl=-5 and Pal=0 cmH2O)
  • Target PAPm is 15 mmHg at a CO of 6.45 l/min and LAP of 5 mmHg

If you want to calibrate the model to represent a single lung, for the same size subject, you will have to enter a new target PAPm for a given CO (the appropriate flow for a single lung) and LAP. If you want to calibrate the model to represent a certain pediatric population, you have to enter an average height and weight, and an average lung height that represents that population. Notice that normal lung compliance is adjusted automatically. You also have to enter a target PAPm for a given CO and LAP.

You can re-calibrate the model at any time to represent different physiological and/or pathophysiological scenarios.

To simulate any scenario in an adult subject, you can just accept the default calibration (which is stored in a database file). The last performed calibration is the one that is stored and used with subsequent runs of the model. You do not need to recalibrate the model prior to each use unless you are simulating a specific conditions and/or a different species other than humans.

The calibration factors can be reset to the default ones at any time from the "Calibration" window.

The "Calibration" window is accessible under the "Window" menu option.

You have the option of running the program for a single set of input variables or for a range of input variables. The latter option is usually used to generate data for graphing.

In the sidebar (Input Model Parameters), single variables can be entered directly. To enter a range, right click on the input variable and select "Set Range". A popup window allowing you to set minimum, maximum and step values. The sequence of values that will be used in the calculations is immediately shown. To reset a range value back to a single value modify the minimum and/or maximum such as the range only contains a single value (minmum = maximum).

Because of the large number of vessels, some vessels may not be apparent on the first screen. To see individual vessels and their characteristics you can zoom in and out using either the "Mouse Wheel" in combination with the "Ctrl" button or by using the "+/-" button on the keyboard.

To modify any vessel characteristics, double click on the vessel. A window pops up with the parameters that can be modified.

To run the model, press the "Calculate..." button.
